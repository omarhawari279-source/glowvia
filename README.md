# 🌸 Glowvia - E-commerce Website

## 📌 Description du projet

Glowvia est un site web e-commerce dédié à la vente de produits cosmétiques naturels et premium.
Le site permet aux utilisateurs de découvrir les produits, consulter les détails et naviguer facilement entre les différentes pages.

Ce projet a été réalisé dans le cadre du module **Problem Solving (DevOps)** afin de pratiquer l’utilisation de **Git et GitHub** dans un environnement collaboratif.

## ⚠️ Gestion des conflits

Un conflit Git a été simulé lors de la modification du fichier CSS.

### ✔️ Cause

Deux membres ont modifié la même partie du fichier style.css

### ✔️ Solution

Le conflit a été résolu manuellement en choisissant la bonne version du code, puis validé avec un commit.
